Plugin generates lyrics in polish language. When You insert shortcode [lyric], it generates random computer lyrics.

== Description ==
There is only one feature - generate lyric. Lyric is generated by computer. 

== Installation ==
1. Upload \"test-plugin.php\" to the \"/wp-content/plugins/\" directory.
1. Activate the plugin through the \"Plugins\" menu in WordPress.
1. Place \"do_action( \'plugin_name_hook\' );\" in your templates.
